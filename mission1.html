MISSION 1 : Coordination Administrative

Pilotage opérationnel quotidien de l'activité terrain

Contexte et périmètre

Chez Starsnett Services, entreprise de nettoyage professionnel intervenant sur 167 sites clients en Île-de-France avec 28 agents terrain, j'assure la coordination administrative complète de l'activité opérationnelle. Cette mission centrale implique la gestion quotidienne des plannings, l'interface avec le système de paie, la coordination des équipes terrain et la réponse aux sollicitations internes et externes.

Application développée : "Qui est où ?"

Face à la complexité croissante de la gestion de 167 sites et 28 agents, j'ai conçu et développé une application web complète permettant de centraliser toutes les informations opérationnelles. Cette Progressive Web App (PWA) est utilisée quotidiennement par 4 utilisateurs (direction, comptable, responsables terrain) et constitue l'outil central de pilotage de l'activité.

Modules fonctionnels (6)

  • Planning quotidien : affectations du jour
  • Gestion sites : 167 fiches détaillées
  • Gestion agents : 28 profils complets
  • Module vitrerie : 35 sites spécialisés
  • Indicateurs comptables : CA, marges
  • Authentification sécurisée par code PIN

Caractéristiques techniques

  • Technologies : HTML5, CSS3, JavaScript vanilla
  • Stockage local + cloud JSONBin (sync optionnelle)
  • Interface responsive (mobile + desktop)
  • Données temps réel accessibles partout
  • Architecture modulaire évolutive

Impact mesurable sur l'organisation

La digitalisation de la coordination administrative a permis une transformation profonde de l'efficacité opérationnelle. Les gains mesurés démontrent la valeur ajoutée de cette mission.

Gains d'efficacité

  • -95% de temps de réponse aux sollicitations (consultation planning instantanée vs recherche manuelle 15-20 min)
  • -80% de sollicitations répétitives grâce à l'autonomie des utilisateurs
  • 100% des données centralisées dans un outil unique accessible partout
  • Zéro erreur de planification depuis mise en production
